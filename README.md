## Installation

### Prerequisites
- Node.js >= 18
- npm or yarn

### Setup

1. Clone the repo
   ```sh
   git clone https://github.com/elfatairy/Portfolio-v2.git
   cd Portfolio-v2
   ```

2. Install dependencies
   ```sh
   npm install
   ```

3. Start the development server
   ```sh
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

### Building for Production
```sh
npm run build
npm run preview
```
